REST API統合

YCBM REST APIを使用して、プログラム的に予約を管理できます。

// 例: 予約一覧の取得
fetch('https://api.youcanbook.me/v1/ACCOUNT_ID/profiles/PROFILE_ID/bookings', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
})
.then(response => response.json())
.then(data => console.log(data));

必要な情報

  • アカウントID
  • プロフィールID
  • APIキー（ダッシュボードから取得）

詳細は YCBM API ドキュメントを参照してください。
